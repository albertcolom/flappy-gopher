Flappy Gopher
==============

### Requirements ###
- [GO 1.18](https://go.dev/)
- [SDL2](https://libsdl.org/download-2.0.php)

### Installation ###
Clone the repository
```sh
$ git clone git@github.com:albertcolom/flappy-gopher.git
```
Install dependencies
```sh
$ go mod vendor
```
Run
```sh
$ go run cmd/main.go
```
